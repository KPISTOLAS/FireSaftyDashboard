üîß 5. TROUBLESHOOTING
=====================

Common Issues and Solutions
--------------------------

Issue: "python is not recognized"
Solution: Use full path to Python
  "C:\Users\kkako.DESKTOP-JTTG40I\AppData\Local\Programs\Python\Python313\python.exe" run.py

Issue: "pip is not recognized"
Solution: Use Python's built-in pip
  python -m pip install -r requirements.txt

Issue: "ModuleNotFoundError"
Solution: Install missing dependencies
  pip install -r requirements.txt

Issue: "Port already in use"
Solution: Change port in .env file or use different port
  PORT=5001 python run.py

Issue: "Database connection failed"
Solution: Application will use mock data automatically
  - Check your .env file for correct database credentials
  - Or leave default values for testing

Issue: "Permission denied"
Solution: Run as administrator or use user installation
  pip install --user -r requirements.txt
